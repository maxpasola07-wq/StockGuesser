# Stock Signal Dashboard V1 Connected

This Next.js project:
- pulls free best-effort stock data using yahoo-finance2
- generates simple rule-based signals
- supports manual approval / rejection of paper trades
- stores paper trades in browser localStorage

## Important
- Free data can be delayed, incomplete, or rate-limited.
- This is not guaranteed-profit software.
- Version 1 is for research and simulation.

## Run locally
npm install
npm run dev

## Deploy
Push to GitHub, then import into Vercel.
